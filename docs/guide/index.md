## 主要功能

ZZFrame 提供了开箱即用的企业级功能：

- **用户管理** - 完整的用户增删改查功能
- **角色管理** - 灵活的角色权限配置
- **菜单管理** - 动态菜单和权限控制
- **登录日志** - 记录用户登录行为
- **权限控制** - 基于 Casbin 的 RBAC 权限模型
