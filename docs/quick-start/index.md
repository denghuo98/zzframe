# 快速开始

欢迎使用 ZZFrame，这是一个基于 GoFrame 的高性能 Web 开发框架，提供了完整的后台管理功能和丰富的组件库。

## 特性

- **高性能**: 基于 GoFrame v2 框架，提供卓越的性能表现
- **权限管理**: 集成 Casbin 权限控制，支持细粒度的 RBAC 权限管理
- **数据缓存**: 支持 Redis 和文件缓存，提升数据访问速度
- **消息队列**: 内置磁盘队列，支持异步任务处理
- **日志系统**: 完整的操作日志记录和查询功能
- **文件存储**: 支持多种存储方式，便于扩展
- **验证码**: 内置图形验证码，提升安全性
- **加密工具**: 提供多种加密算法，保护敏感数据
- **地理位置**: 基于 IP 的地理位置查询
